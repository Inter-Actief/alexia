## Projectstructuur

 * alexia/ - Alle Python-code
 * assets/ - CSS/Webfonts/Images/Javascript
 * bin/ - Hulpscripts
 * config/ - Configuratie voor de environment
 * locale/ - Vertalingen
 * templates/ - Templates
